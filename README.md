# TC-SSDA
The official code of our paper: Target-Calibrated Selection and Source Assistance for Label-Efficient Driving Scene Segmentation.

The code will be made public after acceptance.

## Abstract:
Dense annotation makes the deployment of semantic segmentation in new driving domains expensive. Unsupervised domain adaptation (UDA) removes target annotation but can retain source-biased decision boundaries, whereas target-only semi-supervised learning introduces target anchors but discards labeled source data and is highly sensitive to the selected subset at extreme budgets. We formulate source-available semi-supervised domain adaptation (SSDA) as two coupled problems: selecting which target images to annotate and calibrating how source supervision is used after annotation. We propose TC-SSDA, a two-stage framework for this setting. Stage 1 constructs a one-shot target subset by combining perspective-view-aware DINOv3 embeddings with density--diversity coverage and UDA-teacher estimates of semantic reliability and rare-class utility. Stage 2 separates source supervision, target supervision, source--target mixing, and target-only semi-supervision in a four-branch objective. Target-dominant mixing controls the global source--target balance, while target-deficit class routing restricts source mixing to classes that address current target weaknesses. An optional progressive variant reallocates part of the same annotation budget under large synthetic-to-real shifts and is reported separately because it adds an inference and annotation round. Across five synthetic-to-real and real-to-real driving settings, TC-SSDA matches or exceeds the target-only full-supervision reference using 1/8 of Cityscapes labels, 1/16 of ACDC or MUSES labels, and 1/64 of Mapillary labels. Extensive ablations distinguish target acquisition from UDA initialization and show that training-time source calibration remains complementary to a strong UDA prior.
